# Οδηγίες Πρόσβασης για το BC-FL Platform

---

## Προαπαιτούμενα


* **Docker Desktop**: Εγκατεστημένο (σε Windows ή Mac) και να βρίσκεται ανοιχτό.
* **Git**: Download ως ZIP από το GitHub για κατέβασμα του κώδικα.

---

## 1. Κατέβασμα του Project

**Βήμα 1** 

Άνοιξε το Docker Desktop και περίμενε να εμφανιστεί: *Docker is running*.

**Βήμα 2** 

Άνοιγμα PowerShell / Terminal

**Βήμα 3**

Μετάβαση στον φάκελο που θέλετε να βρίσκεται το project

```
cd Desktop
```

**Βήμα 4**

Κατέβασμα αποθετηρίου από το GitHub

```
git clone https://github.com/Angel-fa/bc-fl-platform.git
```

**Βήμα 5**

Είσοδος στον φάκελο του project

```
cd bc-fl-platform
```

---

## 2. Εκκίνηση Πλατφόρμας με Docker

Μέσα στον φάκελο του project, εκτελέστε:

```
docker compose up -d --build
```

---

## 3. Έλεγχος ότι όλα ξεκίνησαν σωστά

```
docker compose ps
```

---

## 4. Πρόσβαση στην πλατφόρμα

* **UI/Streamlit:**  http://localhost:8501

---

## 5. Υπόδειγμα σεναρίου λειτουργίας

1. Ανοίξτε το UI: http://localhost:8501


2. Προχωρήστε σε **Register** χρήστη:

   * Role: **Hospital**
   * Invite code: HOSPITAL2026

   
3. Ύστερα προχωρήστε σε **Login**



4. Στην ενότητα **Datasets**:

   * Upload αρχείο CSV/Excel
   * Create descriptor
   * Validate
   * Επιλέξτε exposed features


5. Ύστερα μπορείτε να δημιούργησε χρήστη με **Role: Researcher** και να υποβάλετε **Access Request**


6. Στη συνέχεια με **Role: Hospital** επιβεβαιώστε με **approve**.


7. Ως Researcher μεταβείτε:

   * Federated Jobs → Create job → Start job
   * Επιλέξτε την επιθυμητή προβολή για τα αποτελέσματά σας

---

## 6. Τερματισμός Πλατφόρμας

```
docker compose down
```

---

## 9. Κάθε Επόμενη Χρήση μόνο με 2 βήματα

1. Άνοιγμα Docker Desktop
2. Στο PowerShell/Terminal:

```
cd Desktop\bc-fl-platform
docker compose up
```
